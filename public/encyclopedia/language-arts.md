# Language Arts

Study grammar, literature, writing, and communication skills.

## Knowledge Base

### Core Concepts
- Grammar: Syntax, punctuation, parts of speech
- Literature: Fiction, poetry, drama, literary analysis
- Writing: Essay composition, creative writing, technical writing
- Rhetoric: Persuasion, argumentation, public speaking
- Reading Comprehension: Analysis, interpretation, critical thinking
- Linguistics: Language structure, phonetics, semantics

### Fields of Work
- **Writer/Author**: Create books, articles, stories
- **Editor**: Review and improve written content
- **Journalist**: Report news, conduct interviews
- **Copywriter**: Write advertising, marketing copy
- **Technical Writer**: Create manuals, documentation
- **English Teacher**: Educate students in language arts
- **Literary Agent**: Represent authors, negotiate publishing deals
- **Content Strategist**: Plan and manage content for organizations
